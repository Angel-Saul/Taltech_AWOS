# Taltech

## Diseño de Interfaz (UI/UX)

El diseño de la interfaz del sistema Taltech fue desarrollado utilizando la herramienta de diseño Figma, permitiendo estructurar visualmente la solución antes de su implementación.

Se trabajó bajo un enfoque progresivo de diseño, contemplando las siguientes fases:

### Sketches

Se realizaron bocetos iniciales con el objetivo de definir la estructura general del sistema, distribución de elementos y flujo básico de navegación.

---

### Wireframes

Se desarrollaron wireframes para establecer la jerarquía de la información y organización de los componentes en cada vista del sistema.

---

### Mockups

Se diseñaron mockups visuales que representan la apariencia final del sistema, incluyendo estilos, colores, tipografía y elementos gráficos.

---

### Prototipo Navegacional

Se construyó un prototipo interactivo que permite simular la navegación entre las distintas pantallas del sistema, facilitando la validación del flujo de usuario.

---

## Propósito

El uso de herramientas de diseño permitió:

* Visualizar el sistema antes de su desarrollo
* Detectar posibles errores de usabilidad
* Mejorar la experiencia del usuario
* Garantizar coherencia entre diseño e implementación
